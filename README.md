scrum_demo_angular_rails_config
===============================

A demo for the curriculum material covering setting up Angular with Rails.


## Getting started

1. Clone the repo
1. `cd` into the project
1. Run `bundle install`
1. Run `rails s`
1. Open browser to `http://localhost:3000`
